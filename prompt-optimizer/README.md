# Prompt Optimizer

A microservice for prompt optimization using DSPy framework.

## Features

- DSPy-based prompt optimization
- RESTful API endpoints
- Clean Architecture pattern
- Docker support

## Installation

```bash
uv sync
```

## Usage

```bash
uv run dev
```

The service will be available at `http://localhost:9090`

## Development

```bash
uv sync --group dev
uv run dev
```
